# hashnodebot


# How to run ?
Make sure you have working golang installation

` go run main.go <flags> `
```
Usage:
  -debug
    	turn debug bode on/off (default true)
  -port string
    	port to listen (default "80")
  -token string
    	telegram bot token
  -webhookBaseURL string
    	Base URL for webhook
      
 ```
